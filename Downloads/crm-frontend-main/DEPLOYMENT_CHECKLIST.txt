→ Verifique:
  [ ] .htaccess existe em /public_html/?

→ Verifique:
  [ ] Arquivo .css existe em assets/?

  [ ] Tamanho do arquivo é ~60KB?

→ Verifique:
  [ ] Arquivo .js existe em assets/?
  [ ] Caminho em index.html está correto?
  [ ] Status 200 no Network (DevTools)
  [ ] Console mostra erros específicos?
  [ ] Tamanho do arquivo é ~1.6MB?
